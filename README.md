# 象棋辅助工具 (Xiangqi Assistant)

基于屏幕视觉识别 + Pikafish 引擎的智能中国象棋辅助工具。

## 功能特点

- 🎯 **实时分析**：自动识别屏幕棋盘，实时分析最优走法
- 🧠 **强大引擎**：内置开源顶级引擎 Pikafish（NNUE 神经网络，棋力极强）
- 🔄 **随时开关**：按 `F1` 一键开启/关闭辅助，对局中随时切换
- 💡 **文字建议**：半透明悬浮窗显示推荐走法、局面评分、搜索深度
- ⚙️ **难度可调**：通过思考时间控制 AI 强度
- 🖥️ **后台运行**：最小化到系统托盘，不干扰正常对局

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 运行

```bash
python main.py
```

### 3. 使用

1. 打开你的象棋软件（天天象棋、JJ象棋、网页版等）
2. 启动本工具
3. 按 `F1` 开启辅助模式
4. 首次使用按 `F2` 校准棋盘位置
5. 系统会自动分析并给出走法建议

## 界面说明

| 组件 | 说明 |
|------|------|
| **悬浮窗** | 半透明窗口，显示推荐走法、局面评分、搜索深度、推演变例 |
| **主窗口** | 设置引擎参数、棋盘校准、热键配置 |
| **系统托盘** | 右键托盘图标可切换状态、显示主窗口、退出 |

## 操作热键

| 热键 | 功能 |
|------|------|
| `Z` | 开启/关闭辅助 |
| `X` | 校准棋盘位置 |
| `C` | 手动截图识别（调试） |

## 项目结构

```
xiangqi_tool/
├── main.py                    # 主入口
├── config.py                  # 配置管理
├── requirements.txt           # 依赖清单
├── pikafish.exe              # Pikafish 引擎
├── pikafish.nnue             # 神经网络权重
│
├── engine/
│   └── pikafish_engine.py    # UCI 引擎通信
├── vision/
│   └── board_recognition.py  # 棋盘视觉识别 (OpenCV)
├── core/
│   ├── state_machine.py      # 状态机管理
│   ├── hotkey_manager.py     # 全局热键
│   └── converter.py          # 坐标/棋谱转换
└── ui/
    ├── overlay.py            # 透明悬浮窗
    ├── main_window.py        # 设置主窗口
    └── tray.py               # 系统托盘
```

## 注意事项

- 本工具使用**视觉识别**截取屏幕画面，兼容所有象棋软件
- 仅提供**文字建议**，不自动操作鼠标键盘，无封号风险
- 建议在单机练习或线下对弈中使用
- 引擎分析约消耗 1-2 个 CPU 核心
